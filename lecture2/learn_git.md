# Git Learning Resources

**All Videos:** https://git-scm.com/videos

## Getting Started
1. Watch the videos in order from top to bottom
2. Keep the cheat sheet nearby as you practice

## Video Tutorials

The following videos provide an excellent introduction to Git and version control. Watch them in order for a comprehensive understanding.

### 1. What is Git?
- **Description:** Learn the fundamentals of Git, understanding what it is and why it's essential for software development and collaboration.
- **Link:** https://git-scm.com/video/what-is-git

### 2. What is Version Control?
- **Description:** Understand the concepts of version control systems, how they track changes, and why they're critical for any development project.
- **Link:** https://git-scm.com/video/what-is-version-control

### 3. Quick Wins with Git
- **Description:** Discover practical and immediately useful Git commands and workflows.
- **Link:** https://git-scm.com/video/quick-wins

### 4. Get Going with Git
- **Description:** Walks you through the practical steps to start using Git in your projects.
- **Link:** https://git-scm.com/video/get-going
